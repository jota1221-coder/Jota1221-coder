## 🌟 Featured Projects

### 🌌 [StarMap BA — Dark Sky Map for Buenos Aires Province](https://github.com/jota1221-coder/starmap-ba)
Web app for amateur astronomers: finds the best stargazing spots in Buenos Aires Province combining real light pollution data (NASA VIIRS satellite), live weather forecasting, moon phase calculation, and a scoring model that tells you whether tonight is a good night to observe — and why.

**What makes it interesting:** the scoring engine is a pure function with 23 tests, the astronomy calculations run locally (no external API), and the light pollution data was processed from raw satellite imagery.

**Stack:** Next.js 16 · TypeScript · Prisma · PostgreSQL (Neon) · Leaflet · astronomy-engine · Open-Meteo
